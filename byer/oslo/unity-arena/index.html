Unity Arena

Praktisk info om Unity Arena på Fornebu: adresse, kapasitet, parkering, kollektivtransport og billetter. Nedenfor finner du spørsmål og svar om arenaen.

Praktisk info

Adresse, kapasitet og transport

Adresse
Unity Arena adresse er John Strandruds vei 16, 1366 Fornebu. Arenaen ligger på Fornebu like vest for Oslo.
Kapasitet
Unity Arena kapasitet er rundt 15 000 ved konserter. Kapasiteten varierer med sceneoppsett.
Parkering
Unity Arena parkering: Stor parkeringsplass ved arenaen. Betaling gjelder. Ved store arrangementer kan det bli travelt – kom i god tid eller bruk kollektivtransport. Aktuelle priser og info finner du på Unity Arenas nettside.
Kollektivtransport
Kollektivtransport til Unity Arena: Buss fra Oslo S og Fornebu. Sjekk Ruters reiseplanlegger for ruter og avgangstider. Kollektiv anbefales ved store konserter.
Billetter og konserter
Unity Arena billetter kjøpes hos Ticketmaster, Eventim og andre anerkjente billettselskaper. Kommende konserter og program vises på arenaens offisielle nettside.
Salkart og innganger
Salkart og innganger Unity Arena: Hvilken inngang du skal bruke står på billetten (blokk/seksjon). Salkart og detaljer finner du ved billettkjøp og på arenaens nettside.
