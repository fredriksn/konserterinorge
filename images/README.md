# Bilder for Konserter i Norge

## SoMe-forsidebilde (Open Graph / Twitter)

Legg inn en fil som heter **og-default.jpg** i denne mappen.

- **Anbefalt størrelse:** 1200 × 630 piksler
- **Format:** JPG eller PNG
- **Bruk:** Når noen deler forsiden (eller sider uten eget bilde) på Facebook, Twitter/X, LinkedIn eller i meldinger, vises dette bildet.

Bildet bør være lesbart og representere nettsiden (f.eks. logo, konsertmotiv eller en enkel grafikk med «Konserter i Norge»).

Hvis filen mangler, vil noen plattformer vise ingen forhåndsvisning eller bare tittel og beskrivelse.
